## ğŸ” What Each Step Actually Does

### Step 1: Checkout
```bash
# Behind the scenes:
git clone https://github.com/YOUR_REPO.git
cd YOUR_REPO
git checkout COMMIT_SHA
```

### Step 2: Setup Java
```bash
# Behind the scenes:
download temurin-jdk-17
install to /runner/jdk-17
export JAVA_HOME=/runner/jdk-17
check if maven cache exists
  if exists: restore from cache
  if not: prepare to download dependencies
```

### Step 3: Build
```bash
# Actual command:
./mvnw clean install

# What happens:
1. Delete target/ folder (clean)
2. Download dependencies to ~/.m2/repository
3. Compile src/main/java â†’ target/classes
4. Compile src/test/java â†’ target/test-classes
5. Run tests
6. Create JAR file â†’ target/demo-0.0.1-SNAPSHOT.jar
```

### Step 4: Test
```bash
# Actual command:
./mvnw test

# What happens:
1. Run all @Test annotated methods
2. Generate XML reports â†’ target/surefire-reports/*.xml
3. Exit code 0 (success) or 1 (failure)
```

### Step 5: Publish
```bash
# What happens:
Read target/surefire-reports/*.xml
Parse test results
Create GitHub check annotation
Display in PR/commit status
```

### Step 6: Upload
```bash
# What happens:
Zip target/surefire-reports/
Upload to GitHub Actions artifact storage
Set 30-day retention policy
Make downloadable from Actions tab
```
